MultivariateTests.jl

MultivariateTests is a Julia package that extends the framework provided by the HypothesisTests to include functionality for multivariate statistics.

Functionality

  • Measures of dispersion
  • Partial correlation
  • Hotelling's $T^2$ test
  • Bartlett's test

Acknowledgements

Initial work on this package by Alex Arslan was part of a research project for the master's program in applied statistics at Penn State University. Alex thanks Andreas Noack, Jiahao Chen, Moritz Schauer, and Simon Byrne for initial feedback and discussion in the early stages of development.
